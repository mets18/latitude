Summary: Weather by Latitude

The purpose of this project was to assess the impact that latitude has on weather using 4 different measures: temperature, humidity, cloudiness, and wind speed. We used th OpenWeatherMap API to draw data from over 500 cities.

After assembling the data we used Matplotlib to visualize the effect that latitude had on the weather using the forementioned 4 factors. This site provides the source data and visualizations used to explain our analysis and some interpretation on each page.
